章節 49 拉取共享檔案庫的修改

學習目標

  • 了解如何從共享的檔案庫拉取修改。

快速跳到複製的檔案庫，並讓我們拉下剛才送交到共享檔案庫的修改。

執行:

cd ../cloned_hello

注意: 現在位於 cloned_hello 檔案庫。

繼續…

執行:

git remote add shared ../hello.git
git branch --track shared master
git pull shared master
cat README
